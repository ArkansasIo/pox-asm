QUICK START:
------------
1. mkdir build && cd build
2. cmake ..
3. cmake --build . -j
4. Run demos: ./pox8_sim or ./pox64_demo

FEATURES:
---------
- Virtual 8-bit CPU with custom ISA
- 60+ sprites (8x8 and 16x16) for 128x64 LCD
- Memory management (arena, pool, ring buffer)
- Deterministic procedural generation
- Asset packing system
- Binary tools (assembler, hex viewer, dumper)
- Dual target: 8-bit firmware + 64-bit engine
